# 🚜 3D Farm

基于 [Three.js](https://threejs.org) 构建的交互式 3D 农场场景，在浏览器中直接运行，无需任何构建工具。

## ✨ 特性

- **🏗️ 丰富场景** — 谷仓、筒仓、农舍、水井、干草捆、稻草人、马车、狗屋等
- **🌾 多种作物** — 玉米、小麦、南瓜、番茄四种作物
- **🌳 自然环境** — 树木、灌木、云朵、飞鸟、地形起伏
- **🎥 上帝视角** — 拖拽旋转、滚轮缩放、WASD 平移
- **🚶 第一人称** — 按 `F` 切换，WASD 移动、鼠标环顾、Shift 加速
- **🏃 碰撞检测** — 无法穿墙穿树，地形跟随
- **🌤️ 动态效果** — 实时阴影、作物摇摆、云朵飘移、飞鸟盘旋、小狗走动

## 🎮 操作

| 按键 | 上帝视角 | 第一人称 |
|------|----------|----------|
| 鼠标拖拽 | 旋转视角 | 环顾四周 |
| 滚轮 | 缩放 | — |
| `W` `A` `S` `D` | 平移视角 | 前后左右移动 |
| `F` | 切换到第一人称 | 返回上帝视角 |
| `Shift` | — | 加速跑动 |
| `Esc` | — | 退出第一人称 |

## 🚀 运行

直接在浏览器中打开 `index.html` 即可，所有依赖通过 CDN 加载。

或者使用任意静态服务器：

```bash
python -m http.server 8080
# 访问 http://localhost:8080
```

## 🛠️ 技术

- **Three.js** — 3D 渲染引擎
- **OrbitControls** — 轨道相机控制
- **Pointer Lock API** — 第一人称鼠标环顾
- **零依赖构建** — 单 HTML 文件，CDN importmap 加载
